# Remote Job Bot Pro

The ultimate automated remote job hunting toolkit that scans 8 job boards simultaneously, generates tailored resumes, writes cover letters, and tracks your applications - all in a single HTML file.

## 🎯 What It Does

Remote Job Bot Pro is a comprehensive job hunting automation tool designed for serious remote job seekers. Instead of manually checking multiple job boards every day, this tool does the heavy lifting for you.

### Core Functionality

**🔍 Parallel Job Scanning**
- Scans 8 major remote job boards simultaneously every 15 minutes
- Completes full scan in ~15 seconds using parallel fetching
- Auto-rescans continuously while the page is open
- Sends browser notifications when new jobs appear

**🎯 Smart Job Matching**
- Scores every job 0-100% based on your target roles and skills
- Highlights best matches with 👑 and ⭐ badges
- Filters by salary ranges and specific keywords
- Ranks results by relevance to your profile

**🌍 Global Currency Support**
- Filter jobs by any of 83 world currencies
- Perfect for international job seekers
- Supports NGN, GHS, KES, USD, GBP, AED, PHP, INR and more
- Built specifically for African and global remote workers

**📄 Resume Generator**
- Fill in your details once (no LinkedIn required)
- Up to 4 work experience blocks
- Get match scores showing keyword coverage for any job
- Export as professional resume format

**✉️ Cover Letter Writer**
- Paste any job description and generate instant cover letters
- Personalize with hiring manager's name
- Set your own availability statement
- Professional tone and formatting

**📊 Application Tracker**
- Track every job with 6 status options
- Includes "Withdrawn" and "Ghosted" statuses
- Edit notes inline for each application
- Filter by status, search by company
- Export all data as CSV

**⭐ Save & Bookmark Jobs**
- Star any job to save it to the Saved tab
- Jobs persist across browser sessions
- Copy URLs with one click
- Export bookmarks as CSV

**💾 Data Backup & Restore**
- Export everything as single JSON file
- Includes jobs, tracker data, bookmarks, resume, settings
- Restore your complete setup on any device
- Your data never leaves your browser

**🔧 Apply Toolkit**
- Pre-application checklist with persistent states
- One-tap links to all job boards
- Fiverr gig search links
- No more manually typing URLs daily
